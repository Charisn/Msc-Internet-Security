Sample Course Selection Form

Το παραπάνω σκίτσο παρέχει ένα δείγμα πίνακα για την επιλογή των μαθημάτων που θα συμπεριληφθούν στην αναφορά. Αφού επιλεγούν τα μαθήματα από τον πίνακα, η αναφορά μπορεί να δημιουργηθεί κάνοντας κλικ στο παρακάτω κουμπί.

Μπορείτε να κάνετε κλικ στο κουμπί παρακάτω για να προβάλετε ένα σκίτσο της αναφοράς που αναμένεται να αναπτύξετε.
